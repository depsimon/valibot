# minLength

Creates a pipeline validation action that validates the length of a string or an array.

```ts
const Validation = minLength<TInput, TRequirement>(requirement, message);
```

## Generics

- `TInput` <Property {...properties.TInput} />
- `TRequirement` <Property {...properties.TRequirement} />

## Parameters

- `requirement` <Property {...properties.requirement} />
- `message` <Property {...properties.message} />

### Explanation

With `minLength` you can validate the length of a string. If the input does not match the `requirement`, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />

## Examples

The following examples show how `minLength` can be used.

### String schema

Schema to validate a string with a minimum of 8 characters.

```ts
const StringSchema = string([
  minLength(8, 'The string must contain at least 8 characters.'),
]);
```

### Array schema

Schema to validate an array with a minimum of 100 numbers.

```ts
const ArraySchema = array(string(), [
  minLength(64, 'The array must contain at least 100 numbers.'),
]);
```
